# AGENTS.md

## 项目背景

行测（行政职业能力测验）备考资料整理项目。包含常识判断、言语理解、数量关系、判断推理、资料分析五大模块。

## 全局规则

- 所有思考、回复、注释等内容必须使用中文。
- 代码和文档遵循简洁原则，不加多余注释。
- 文件命名统一使用小写字母 + 下划线。
- 所有文本文件使用 UTF-8 编码。
